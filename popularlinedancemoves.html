Popular Line Dance Moves


Below is a list of some of the most popular line dance moves. These help build up steps in many line dances and knowing them should help you get a better idea of how to read stepsheets and learn the different steps of line dancing.


My reccomendation is to learn a few line dances then come back to review these. From my experience, getting used to the moves then learning the names can be really helpful.
